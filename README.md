# AI_Study_Buddy
# 📚 AI Study Buddy

**Your personal AI-powered study companion** — designed to help students organize, learn, and prepare smarter.  

With **AI Study Buddy**, you can:  
✅ Summarize long study materials  
✅ Create instant flashcards for revision  
✅ Plan exams with reminders  
✅ Track study sessions & subjects  
✅ Save feedback & generate reports  

---

## 🚀 Features

- **🔐 User Login** – Each student has their own account.  
- **📝 Summarizer** – Paste text and get concise summaries.  
- **💡 Flashcards** – Generate question-answer cards instantly.  
- **📅 Exam Planner** – Schedule and manage upcoming exams.  
- **📊 Study Tracker** – Record and visualize your study sessions.  
- **📑 Reports** – Save and revisit your performance reports.  
- **💬 Feedback System** – Share suggestions and ratings.  

---

## 🛠️ Tech Stack

- **Frontend:** [Streamlit](https://streamlit.io/)  
- **Backend:** Python (SQLite for storage)  
- **AI Models:** Hugging Face Transformers for summarization  
- **Visualization:** Matplotlib & Seaborn  
- **Auth:** JSON-based login system

## ⚡ Installation & Run

1. **Clone the repo**
   ```bash
   git clone <repo-url>
   cd AI_Study_Buddy


2. **Install the required libraries:**
- **The project's dependencies are listed in requirements.txt.**

   **Install them using pip:**

   **pip install -r requirements.txt**

3. **Run the application:**

   **streamlit run Main.py**

   Your application will open in your web browser. 

## 🧠 How it Works
-**Login to your account.**

-**Pick a tool from the sidebar (Summarizer, Flashcards, etc.).**

-**Use AI-powered tools to save time and study smarter.**

-**Your data is stored securely and is available anytime you log in.**
